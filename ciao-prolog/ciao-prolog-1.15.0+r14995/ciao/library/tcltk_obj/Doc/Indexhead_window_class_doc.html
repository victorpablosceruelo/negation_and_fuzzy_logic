CLIP Local Manuals (lpdoc-generated)

This is the CLIP group's local site for the manuals for CLIP software packages, generated using the lpdoc documentation generator. These manuals correspond to the versions of these packages which are installed / in development at the CLIP lab, as opposed to the versions being distributed, which are in the public CLIP WWW public and restricted-access (Alpha and Beta) software distribution areas. Note that these manuals can often also be accessed through the standard man command and through the emacs info facilities (M-x info). The latter is often the most practical way to use these manuals on-line.
