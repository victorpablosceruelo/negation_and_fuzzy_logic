1. INTRODUCTION

This scope of the HTTP Client component is an implementation of the HTTP protocol, as defined in RFC 2616 and RFC 2617.

The original source was donated by the Jakarta Slide project.

2. DEPENDENCIES

The HTTP CLient component is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.1 or later)
